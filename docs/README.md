# Superstore Sales Analysis

## Objetivo
Projeto de análise de dados utilizando o dataset Superstore Dataset, disponível no Tableau Public, com foco em:
- Organização de dados em camadas - Medallion Architecture (Bronze, Silver, Gold)
- Qualidade e consistência dos dados
- Análises de vendas e lucratividade
- Visualização final em Power BI

## Estrutura do Projeto
- data/: dados brutos (não versionados);
- sql/: scripts SQL organizados por camada;
- docs/: documentação de qualidade de dados;
power_bi/: visualizações finais (não versionadas)

## Tecnologia
- VSCode + Extensão PostgreSQL
- SQL
- Power BI
- Git
